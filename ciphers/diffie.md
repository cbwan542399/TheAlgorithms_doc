## 案例一：依据一个素数,寻找一个另一可用素数
 
```
def find_primitive(n: int) -> Optional[int]:
    for r in range(1, n):
        li = []
        for x in range(n - 1):
            val = pow(r, x, n)  
            # pow(x, y[, z]), 函数是计算x的y次方，如果z在存在，则再对结果进行取模，
            # 其结果等效于pow(x,y) %z
            if val in li:
                break
            li.append(val)
        else:
            return r
    return None
```
